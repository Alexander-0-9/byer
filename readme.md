#  Værapp

Dette er en enkel værapp som henter sanntids værdata fra OpenWeather API. Brukeren kan søke på en by og få informasjon om temperatur, værbeskrivelse og minimum/maximum temperatur.

---

##  Funksjoner
- Søk på by og få værdata
- Viser temperatur i °C
- Viser værbeskrivelse (f.eks. skyet, sol osv.)
- Viser min og maks temperatur
- Søk fungerer både med knapp og Enter
- Søkefeltet tømmes automatisk etter søk

---

##  Teknologier brukt
- HTML
- CSS
- JavaScript
- OpenWeather API

---

##  Hvordan bruke prosjektet
1. Last ned filene
2. Åpne `index.html` i en nettleser
3. Skriv inn en by og trykk Enter eller klikk Søk

---
